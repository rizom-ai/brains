# Relay operating model

Relay is optimized for shared team knowledge rather than personal publishing.

The current POC loop is:

1. Capture notes and links.
2. Preserve conversations as summaries.
3. Maintain a peer-brain directory.
4. Extract topics from the existing corpus.
5. Expose a minimal public knowledge surface when the preset enables it.

Relay full currently adds docs and decks only. Future content should wait for future plugins.
