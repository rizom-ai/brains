# Capture and synthesis

Relay uses existing entity plugins for the first POC:

- notes capture team-local context
- links preserve external references and summaries
- summaries turn conversations into durable memory
- topics synthesize notes, links, summaries, agents, skills, docs, and decks

Draft links are included in topic extraction because private team understanding should not wait for public publication.
