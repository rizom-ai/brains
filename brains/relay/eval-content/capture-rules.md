# Capture rules

- Save raw team observations as notes when the source is human context.
- Save external references as links so extraction can preserve source URLs and summaries.
- Keep peer-brain records in the agent directory before any A2A call.
- Use summaries for conversation memory rather than inventing standalone future-plugin entities.
- Use docs and decks only in the full preset, where they represent existing team knowledge surfaces.
