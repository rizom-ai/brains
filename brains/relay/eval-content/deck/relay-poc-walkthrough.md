# Relay POC

Capture → synthesize → coordinate.

---

# Core loop

- Notes capture team context
- Links preserve external references
- Summaries preserve conversations
- Topics synthesize the current corpus

---

# Coordination

- Agents model peer brains
- Skills describe available capabilities
- A2A calls require approved local agent records

---

# Boundary

Relay is not Rover for teams. Docs and decks are current full-preset surfaces; future-plugin content waits until those plugins exist.
