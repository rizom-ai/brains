# @brains/default-site-content

Default content, templates, and assets for Brain-generated websites.

## Overview

This package provides default themes, templates, styles, and content for sites generated by the site-builder plugin. It includes responsive themes, starter templates, and common assets.

## Features

- Multiple built-in themes
- Responsive templates
- Default styles and layouts
- Sample content and pages
- Common web assets
- Icon sets and fonts
- SEO-optimized templates

## Installation

```bash
bun add @brains/default-site-content
```

## Usage

```typescript
import { 
  getTheme,
  getTemplate,
  getDefaultContent,
} from "@brains/default-site-content";

// Get a theme
const theme = getTheme("minimal");

// Get a template
const template = getTemplate("blog-post");

// Get default content
const content = getDefaultContent("home");
```

## Themes

### Available Themes

```typescript
import { themes } from "@brains/default-site-content";

// Minimal - Clean, minimalist design
const minimal = themes.minimal;

// Blog - Blog-focused layout
const blog = themes.blog;

// Documentation - Technical documentation
const docs = themes.documentation;

// Portfolio - Portfolio/showcase
const portfolio = themes.portfolio;

// Magazine - Magazine/news style
const magazine = themes.magazine;
```

### Theme Structure

```typescript
interface Theme {
  name: string;
  description: string;
  version: string;
  styles: {
    main: string;      // Main CSS
    responsive: string; // Media queries
    print: string;     // Print styles
  };
  templates: {
    layouts: Record<string, string>;
    partials: Record<string, string>;
    pages: Record<string, string>;
  };
  assets: {
    fonts: string[];
    icons: string[];
    images: string[];
  };
  config: ThemeConfig;
}
```

## Templates

### Layout Templates

```handlebars
{{!-- layouts/default.hbs --}}
<!DOCTYPE html>
<html lang="{{lang}}">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>{{title}} - {{site.name}}</title>
  <link rel="stylesheet" href="/assets/css/main.css">
  {{> meta}}
</head>
<body>
  {{> header}}
  <main>
    {{{content}}}
  </main>
  {{> footer}}
  {{> scripts}}
</body>
</html>
```

### Page Templates

## Customization

### Extend Theme

```typescript
import { extendTheme } from "@brains/default-site-content";

const customTheme = extendTheme("minimal", {
  styles: {
    custom: `
      .custom-component {
        /* Custom styles */
      }
    `,
  },
  templates: {
    pages: {
      custom: customPageTemplate,
    },
  },
});
```

### Override Templates

```typescript
import { overrideTemplate } from "@brains/default-site-content";

overrideTemplate("header", `
  <header class="custom-header">
    <!-- Custom header -->
  </header>
`);
```

## Performance

### Optimized Assets

- Compressed images
- Lazy loading support
- Critical CSS inlining
- Font subsetting

### Build Integration

```typescript
import { optimizeAssets } from "@brains/default-site-content/build";

const optimized = await optimizeAssets({
  css: { minify: true, purge: true },
  js: { minify: true, bundle: true },
  images: { compress: true, formats: ["webp"] },
});
```
