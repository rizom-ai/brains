Implementation notes
  1. One combined widget replaces the two registrations in entities/agent-discovery/src/plugin.ts:59–119. Sidebar Skills widget in src/plugins/skill-plugin.ts is left alone — different job (identity, not operator overview).
  2. New renderer AgentNetworkWidget, modeled on PipelineWidget's tab/panel structure. One registration, one data provider, three internal views.
  3. Three views. Overview (default): composition headline + SWOT 2×2, no individual rows. Agents: kind-tabbed cast list (all · professional · team · collective). Skills: tag-aggregated capability inventory; rows are tags, chips are sources (brain + agents).
  4. Shared primitive: entities/agent-discovery/src/lib/coverage.ts. Computed in-memory over current agent + skill entities — no stored graph, no schema change. Exposes: { summary: { totalAgents, coveredSkills, gaps, novelCapabilities, review }, agents: Map<kind, AgentRow[]>, skillsByTag: TagRow[], swot: SwotData }.
  5. SWOT cells derived, not authored. Strengths = tags with ≥ N sources; Weaknesses = local skills with 0/1 coverage; Opportunities = tags present on agents but absent from brain (extensions); Threats = pending-review agents + single-source covered tags. Each cell renders top 2–3 items by magnitude; cell stays present even when empty (kept shape) with a muted "—".
  6. Tag aggregation depends on shared vocabulary. New helper entities/agent-discovery/src/lib/tag-vocabulary.ts: collects current tags across existing skills + agents, passes them into both skill-deriver and AgentGenerationJobHandler prompts as prior art. "Reuse an existing tag when one fits; propose new only when nothing does." No registry entity — the vocabulary is the current corpus, computed at generation time. Tag drift self-corrects on next regeneration.
  7. Agents panels. Sort by skills.length desc, tiebreak discoveredAt desc. Limit 8 per kind. Pill discovered → pill--warn labeled review; approved rows show no pill.
  8. Skills panel. Sort by source count desc, alphabetical tiebreak. Tags carrying only the brain (no agent) render a dashed gap pill in the meta slot. The brain chip uses the accent color to visually distinguish the two levels.
  9. Headline. One Fraunces-set sentence derived from the summary — e.g., "A network of 8 collaborators across 14 capability areas." Italic counts. Second clause carries the sharpest signal (strongest tag, or top unresolved weakness).
  10. Cross-brain federation reuses the same mechanism: each brain's registry is local, but priming converges vocabulary across networked brains over time.
