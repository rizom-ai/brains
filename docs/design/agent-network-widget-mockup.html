Agent directory mockup

One widget, a few tabs — network first.

This mockup treats the directory widget as a compact map of your saved network: who is in it, what they collectively offer, and which contacts stand out. It includes both discovered and approved agents.

What this mockup is testing

  • Whether the overview feels like a network diagnosis rather than a second capabilities view.
  • Whether the capabilities tab feels like offered help lanes, not just relabeled asks.
  • Whether the agents tab needs kind filters to slice the network more intentionally.

Interpretation

The widget is trying to say: "here is the shape of your network and the kinds of help available in it", not: "here is your task queue."
