Your Data, Your Rules

Everything is a plain text file you can read with any editor

M
Markdown, not databases

Every entity — posts, notes, links, decks — lives as a markdown file with frontmatter. Version-controlled with git. Readable without the brain running.

S
Self-hosted, open source

Deploy to your own server with one command. Or run locally. Apache-2.0 licensed. No vendor lock-in, no subscriptions, no fine print.

A
AI model agnostic

Not tied to any single provider. Swap models, combine them, or bring your own. Works with the best AI available today and adapts tomorrow.

Quick Start

Running in under a minute

One package. Three commands. Your brain boots with a website, MCP server, CMS, and chat interface — all in a single process.

Install
$ bun add -g @rizom/brain
Create
$ brain init mybrain --model rover
Run
$ cd mybrain && brain start
✓ Website and CMS on the shared web host
✓ MCP server for assistants and tools
✓ A2A discovery endpoint for agent-to-agent use
✓ Plain-text content you can inspect and version with git
