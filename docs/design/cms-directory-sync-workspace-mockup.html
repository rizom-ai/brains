01 · Ownership

Provider owns truth

directory-sync shapes the browser-safe snapshot, validates actions, and records bounded operational history. CMS only transports and renders it.

02 · Action scope

One manual action

“Sync now” uses the same orchestration as the existing tool. No force push, reset, config editing, or implied rollback.

03 · Attention

Only actionable failures

Quarantined files and failed runs earn a badge. Ordinary watcher activity and a brief dirty working tree do not.

04 · Safety

Sanitized operator data

Show relative paths and a safe remote label—never auth tokens, stack traces, or secret-bearing Git URLs.
