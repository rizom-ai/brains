What changed in rev 2: no cards, no rounded containers, no panel grid. Structure comes from hairline rules, a book-style margin rail, giant folio numbers, and one accent per room (amber / coral / violet). The console aesthetic stays in the console.
How it functions (unchanged): one brain serves everything — essays, gatherings, and products are entities published through the pipeline; the same agent answers "Talk to this brain," exposes the agent card, and serves the canonical lexicon registry. The living-proof line is the demo.
Old domains: rizom.work and rizom.foundation 301 per-path into /work and /foundation; each room's footer band acknowledges the move.
