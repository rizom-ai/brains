How it functions: one brain serves everything. Essays, events, products, and testimonials are entities; the pipeline publishes them; the same brain answers "Talk to this brain", exposes the agent card for peer agents, and serves the canonical ai.rizom.brain.* lexicon registry. The site is the platform's best demo because it *is* the platform.
One theme, three accents: the product/editorial/studio theme variants collapse into the violet-night base with per-room accents — amber (platform), coral (practice), violet (foundation) — carried by nav underline, crumbs, marks, and CTAs.
Old domains: rizom.work and rizom.foundation 301 per-path into /work and /foundation; each room's footer band acknowledges the move so redirected visitors aren't disoriented.
