Notes
  • One shared Preact SVG component serves both surfaces; only the climate differs. Positions come from distance/bearing in the datasource payload — the schema stays presentation-free.
  • All layout randomness (thread bows, rootlets, spores) is seeded — site builds must be byte-identical across runs. Live data replaces the fixtures 1:1.
  • Second-order sightings (Vale via kai.brain, Cairn via north.ops + forge.dev) are the parked Phase 5: light decays with hops, threads route through the introducer, radius stays semantic.
  • Nutrient pulses ride approved threads only; all motion stops under prefers-reduced-motion.
  • Cluster labels derive from members' most frequent skill tags; hovering a mist, label, or member wakes the whole weave.
